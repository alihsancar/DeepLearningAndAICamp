# Deep Learning and AI Camp 

Bu repository, Udemy'deki "Derin Öğrenme Kampı" kursu boyunca kendi yazdığım kodları, çalışma notlarımı ve Jupyter Notebook uygulamalarını içermektedir.

##  İçerik
Bu repo içerisinde ağırlıklı olarak aşağıdaki konular üzerine pratik uygulamalar yer almaktadır:
* Yapay Sinir Ağları (ANN) Temelleri
* Evrişimli Sinir Ağları (CNN) ve Bilgisayarlı Görme
* PyTorch ile Model Eğitimi (Training Loops)
* Veri Ön İşleme (Data Preprocessing) ve Optimizasyon

##  Kullanılan Teknolojiler ve Kütüphaneler
* Python
* PyTorch
* NumPy & Pandas
* Matplotlib & Seaborn
